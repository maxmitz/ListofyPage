Knucklebones


Knucklebones is a fun dice game. You can play it with one or two players. The game idea comes from the game Cult of the Lamb, which is a mini-game in the main game. I liked it a lot and didn't find an iOS version, so I decided to make one myself. I appreciate any feedback and suggestions. Open an Issue in the repository if you find a bug or have an idea to improve the app. The app is programmed in Flutter. Contributions are welcome. Just open a PR.
